How to use

This application provides a collection of movies from The Movie Database with Korean translations. Here are some features you can use:

  • You can search a title
  • You can filter by genre
  • You can click genres underneath cards to filter by genre instead
  • Underneath the main card content is a list of page numbers. You can navigate through the list of movies using these.
  • If you hit the 'x' on the top right of each card, you can remove it
